OAuth User Flow
Step-by-step PKCE flow to get a User Access Token

Step 1 — Generate Auth URL

Generate the authorization URL with PKCE. The user will be redirected here to grant permissions.

Step 2 — Handle Callback

After the user authorizes, Blaze redirects back with a code and state. Paste the full callback URL or just the code here.

Step 3 — Refresh Token

Use the refresh token to get a new access token without re-auth.
